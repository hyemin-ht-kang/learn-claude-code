## 학습 가이드

- **초급** (섹션 01-02): Claude Code의 기본 동작 원리와 도구 시스템
- **중급** (섹션 03-06): 에이전트, 권한, 메모리, 인터페이스의 심화 이해
- **고급** (섹션 07-10): 인프라, 연결성, SDK, 미공개 기능의 전문 지식

각 챕터는 20~40분 분량이며, 섹션 마지막에 인터랙티브 퀴즈가 포함되어 있습니다.
